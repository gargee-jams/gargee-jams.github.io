How it Works

With the use of a Vue component, you can customize the breed of cat that each instance of the conponent shows a slideshow for. It takes one prop, which gives the component the name that is used to find the correct id in the Cat API. The template contains the HTML that will be inserted when the component is used. The data section returns each of the variables. Once the component is created, it calls loadNextImage with the whichCat prop. loadNextImage pre-loads the first 20 images for each breed to display them in the slideshow. It also collects the data for each of the descriptors like the temperament, friendliness levels, and name among other things. The button created in the template is linked to the slideshow function, so each time it is clicked, the slideshow function is executed, and it displays the 10 pre-loaded images. The bar charts are created by making several colored divs that are sized according to their values.
