Getting Started

  1. Make sure you have a GitHub account.
  2. Go to Google Domains, and buy a domain. Basic domains are 12/yr.

    Setting up Your Domain

  1. On Google Domains, go to the side menu and click My Domains.
  2. Select the domain you are using for this course.
  3. Navigate to the side menu and select DNS.
  4. Change the following:
    1. Host Name: nmi
    2. Type: CNAME
    3. TTL: 60
    4. Data: githubUsername.github.io.

    Setting up GitHub Pages

  1. Refer to the following tutorial.

    Adding your Custom Domain

  1. Enter your repository and go to settings.
  2. Click on the Pages tab on the left-hand side.
  3. Scroll until you find Custom Domain
  4. Enter your custom domain without the www. at the beginning, and hit save. The DNS check should begin, and it should soon turn green.
  5. Check the box that says Enforce HTTPS.
  6. Congratulations, your site is now live at nmi.customDomain!

    Using your Site

  1. Open GitHub, and open your repository. Using GitHub Desktop makes this process far more convenient than using the website.
  2. On the main page, there should be an option labeled Open the repository in your external editor.
  3. Make sure your preferred editor is selected, and click open. ex: Visual Studio Code
  4. This should open your repository in the text editor from where you can edit like normal.

    Update your Site

  1. After saving your work on your editor, reopen GitHub Desktop.
  2. On the left-hand side you should see the name(s) of changed files.
    • If only one file has been changed, a summary will not be required, though it is suggested.
    • If multiple files have been chnaged, a brief summary is required before you are allowed to commit.
  3. Hit commit.
  4. At the top of the page, you should now see a button that says Push Origin. Click this button to push changes to your site.
  5. Now, if you go to your website and refresh, your changes should appear. This may take an extra minute or two to load. If you do not see your changes, one of two things has happened:
    1. You need to clear your browsing data to help speed up your updating.
      1. Go to the three-dot menu in the top right corner of Chrome, select More Tools, and then select Clear Browsing Data.
      2. This should bring you to Chrome Settings where you can clear one hour of browsing data. Refreshing your website should now show the changes.
    2. There may be a problem with your HTML/CSS/JS.
