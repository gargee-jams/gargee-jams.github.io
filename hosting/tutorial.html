Getting Started

  1. Make sure you have a GitHub account.
  2. Go to Google Domains, and buy a domain. Basic domains are 12/yr.

      3. Setting up Your Domain

    1. On Google Domains, go to the side menu and click My Domains.
    2. Select the domain you are using for this course.
    3. Navigate to the side menu and select DNS.
    4. Change the following:
      1. Host Name: nmi
      2. Type: CNAME
      3. TTL: 60
      4. Data: githubUsername.github.io.
